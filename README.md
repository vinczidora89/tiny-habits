# habits

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Run json-server to mock the backend database for the API requests
```
json-server --watch src/db/db.json
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
